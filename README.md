# Backend Fundamentals

A personal learning repository for backend development with Node.js and Express.
Every line of code is documented with **why** it's written, not just what it does.

## Structure

| Folder | What you'll learn |
|---|---|
| `js-fundamentals/` | Core and modern JavaScript needed for backend dev |
| `rest-api/` | Building REST APIs with Express — all HTTP methods, middleware, error handling |

## How to use this repo

- Read the `README.md` inside each folder first
- Every file has inline comments explaining the reasoning
- Run examples with `node filename.js` or start the API server as described in `rest-api/README.md`

## Prerequisites

- Node.js installed (v18+ recommended)
- Basic familiarity with terminal

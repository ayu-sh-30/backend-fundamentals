# JavaScript Fundamentals for Backend Development

This section covers JavaScript in three layers — from basics to async patterns.
All topics here are things you WILL use when writing Node.js/Express code.

## Sections

### 01-basics
Core JS you must know before anything else.
- `variables.js` — var vs let vs const, scope, hoisting
- `functions.js` — declarations, expressions, arrow functions
- `loops.js` — for, forEach, map, filter, find, reduce

### 02-modern-js
ES6+ features used heavily in backend code.
- `destructuring.js` — pulling values from objects and arrays cleanly
- `spread-rest.js` — spreading and collecting values
- `arrow-functions.js` — concise syntax and how `this` behaves
- `template-literals.js` — string interpolation and multiline strings
- `modules.js` — require vs import/export

### 03-async
How JavaScript handles operations that take time (file reads, DB calls, API requests).
- `callbacks.js` — the original pattern, and why it gets messy
- `promises.js` — cleaner async handling
- `async-await.js` — the modern, readable way (what you should use)
